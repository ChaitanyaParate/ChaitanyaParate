## Work

**[DeskAI](https://github.com/ChaitanyaParate/Deskai)**
Local-first desktop AI daemon. systemd user service, UNIX socket IPC, streaming LLM
inference via Ollama, offline/online switching. Built in Python on Linux.

**[SE-Attention-Half-UNet](https://github.com/ChaitanyaParate/SE-Attention-Half-UNet)**
Custom medical image segmentation model. Attention gates, Squeeze-and-Excitation blocks,
BCE-Dice hybrid loss, AMP training. Evaluated on Kvasir-SEG polyp dataset.

**[ResNet50 From Scratch](https://github.com/ChaitanyaParate/ResNet50-From-Scratch-Pytorch-Imagenette)**
Full ResNet-50 implementation in PyTorch without torchvision. Trained on Imagenette.
Includes live inference from an ESP32-CAM stream via OpenCV.

**[Brain Tumor Classification](https://github.com/ChaitanyaParate/Modified-Half-UNet-for-Medical-Image-Classification)**
Encoder-only Half-UNet repurposed for binary MRI classification. ~98% accuracy on
Kaggle brain tumor dataset.

**[stm32-hyperloop-embedded](https://github.com/ChaitanyaParate/stm32-hyperloop-embedded)**
STM32 firmware for Hyperloop pod development. Custom Bluetooth bootloader, FreeRTOS
BMS/VCU braking system, 3-phase inverter PWM control, CAN bus. Written in C.
